You are a fact-checking assistant.
Verify the following claim using the provided evidence.

Claim: {claim_text}

Evidence:
{evidence_text}

Task:
Determine if the evidence supports, contradicts, or is insufficient to judge the claim.
Use these definitions:
- SUPPORTED: The evidence is about the same topic and is broadly consistent with the claim's direction. An exact word-for-word match is NOT required — conceptual agreement counts.
- CONTRADICTED: The evidence explicitly states something that directly opposes or disproves the claim.
- INSUFFICIENT: The evidence is about a completely unrelated topic and provides no useful context for the claim.

Important guidance:
- Lean toward SUPPORTED if the evidence discusses the same subject and does not explicitly contradict the claim.
- Only use INSUFFICIENT if the evidence is genuinely off-topic (wrong subject entirely).
- If the claim cites a specific number and the evidence discusses the same metric in the same ballpark, treat this as SUPPORTED.

Output Format:
Return a JSON object with keys: "status", "reasoning".
"status" must be one of: "SUPPORTED", "CONTRADICTED", "INSUFFICIENT".
"reasoning" should be a brief explanation (1 sentence).

Example:
{{
  "status": "CONTRADICTED",
  "reasoning": "The evidence states that crime decreased by 5% in 2022, contradicting the claim of an increase."
}}
